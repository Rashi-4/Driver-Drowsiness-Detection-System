# Driver Drowsiness Detection System

## Overview

A real-time Driver Drowsiness Detection System developed using Computer Vision and Artificial Intelligence.

The system monitors a driver's facial features through a webcam and detects signs of fatigue using:

* Eye Aspect Ratio (EAR)
* Facial Landmark Detection
* Head Nod Detection

Based on the analysis, the driver is classified as:

* Active
* Drowsy
* Sleepy

and appropriate alerts are generated.

## Features

* Real-time face detection
* Facial landmark extraction using Dlib
* Eye Aspect Ratio (EAR) calculation
* Head nod detection
* Active, Drowsy and Sleepy state classification
* Real-time visual alerts
* Low-cost implementation using a webcam

## Technologies Used

* Python
* OpenCV
* Dlib
* NumPy
* Imutils
